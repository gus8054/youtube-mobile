# 목적
Fetch API를 이용해 외부 데이터를 가져와 화면에 렌더링하는 일련의 작업을 수행해보기 위해 제작했다.

# 사용한 기술
* react18+ ( react function components )
* CSS Module
* react-router v6
* vite
* react-icons
* Web Storage API
* Fetch API
* ReactPlayer
* Youtube Data API v3
* uuid
